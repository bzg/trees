#+title: Trees

[[https://intver.org/][https://img.shields.io/badge/versioning-intver.org-blue.svg?style=for-the-badge]]

Generate a static HTML/CSS/JS page that allows users to walk through a decision tree.

#+html: <img src="trees.webp" />

* Install

Assuming [[https://github.com/babashka/bbin][bbin]] is installed:

#+begin_src sh
bbin install io.github.bzg/trees
#+end_src

* Run

You can try it against the =config-example.yml= file provided in this repository:

: ~$ trees config-example.yml

* CSS frameworks

Set =framework= in your config file.  The default is =pico=.

| Framework | Value |
|-----------+-------|
| [[https://picocss.com][Pico 2]]    | =pico=  |
| [[https://purecss.io][Pure CSS]]  | =pure=  |
| [[https://bulma.io][Bulma]]     | =bulma= |

* Themes

When using Pico, you can set a theme with =css-theme=:

#+begin_src yaml
css-theme: "doric"
#+end_src

The =css-theme= option accepts:

- A pico-themes name (e.g. =ashes=, =doric=, =swh=)
- An =https://= URL to a remote CSS file
- A =file:///= URL to a local CSS file
- A local =.css= file path

Available themes from [[https://github.com/bzg/pico-themes][pico-themes]]: =swh=, =org=, =doric=, =lincoln=, =teletype=, =ashes=.

For backward compatibility, =theme-url= and =theme= are still supported.
